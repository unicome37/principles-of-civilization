# 经济学的哲学原理

> **经济不是金钱的流动，而是稀缺条件下，人类对生存机会、协作秩序与未来可能性的持续重组。**

---

## 总论

这部作品回答的不是"怎么学经济学模型"，而是：

**人类为什么会交换、分工、协作、竞争、积累，并由此形成社会结构？**

普通人并不缺"经济活动体验"——人人每天都活在经济学里。缺的是一个能把这些体验重新照亮的解释框架。

经济学本质上不是"钱"的学问，而是**有限条件下如何组织生存**的学问。

---

## 核心问题

- 为什么人会焦虑稀缺？稀缺到底是什么？
- 为什么有些合作越做越大，有些关系一谈钱就碎？
- 为什么分工提升效率，却也制造疏离？
- 为什么市场看起来是自由的，最后却总会形成强者聚集？
- 为什么价格不是"真相"，只是局部条件下的显影？
- 为什么"便宜"有时更贵，"贵"有时反而更稳定？
- 为什么经济增长不等于生活质量增长？
- 为什么有些国家富了却不幸福，有些社会穷了却还有秩序？

---

## 卷目规划

| 卷 | 标题 | 核心问题 | 对应结构 | 状态 |
|----|------|---------|---------|------|
| 第一卷 | 稀缺：为什么人永远觉得不够 | 稀缺不是物质问题，而是存在条件 | 经济的起点 | 📋 规划中 |
| 第二卷 | 选择：为什么每一个"要"都意味着一个"不要" | 选择即放弃，机会成本是经济思维的骨架 | 决策的结构 | 📋 规划中 |
| 第三卷 | 交换：为什么人类一定会发明市场 | 交换不是自利的副产品，而是协作的基本形式 | 协作的起源 | 📋 规划中 |
| 第四卷 | 分工：为什么合作创造财富，却也制造新的不平等 | 分工是效率的引擎，也是结构性依赖的起点 | 秩序的代价 | 📋 规划中 |
| 第五卷 | 价格：为什么价格不是真相，而是局部条件下的信号 | 价格是信息的压缩品，不是价值本身 | 显影与遮蔽 | 📋 规划中 |
| 第六卷 | 制度：为什么规则比人更持久，又比人更脆弱 | 制度是协作的骨架，也是权力的沉积 | 秩序的维持与瓦解 | 📋 规划中 |

## 与自然科学六卷的结构对位

```
自然科学                          经济学
第一卷  变化（运动与规律）    →   第一卷  稀缺（有限性与起点）
第二卷  存在（约束与筛选）    →   第二卷  选择（放弃与机会成本）
第三卷  秩序（耗散与稳态）    →   第三卷  交换（协作的涌现）
第四卷  结合（层级与构型）    →   第四卷  分工（层级与依赖）
第五卷  生命（自维持与演化）  →   第五卷  价格（信号系统的自维持）
第六卷  知识（近似与可靠）    →   第六卷  制度（秩序的近似可靠性）
```

> 自然科学从"世界如何存在"走到"知识如何可靠"；经济学从"人为什么觉得不够"走到"为什么有些秩序能持久"。同一条母线的社会显影。

## 写作方法

遵循统一转译路径：

```
体验 → 困惑 → 结构 → 原理 → 回到生活
```

每一章从普通人的经济体验入口（买东西、找工作、借钱、交税），逐步提升到结构解释，最后回落到人的现实处境。

详见 [方法学说明](../_methodology/从高密度到中高密度.md)

---

## 在整个系列中的位置

经济学的原理处理的是**协作的结构**——人群如何在稀缺中组织交换、分工、竞争与秩序。

它承接自然科学的原理（世界如何存在），为金融学的原理（文明如何穿越未来）做铺垫。

> 如果说自然科学解释了"世界为什么有规律"，经济学就是在解释"人群为什么有秩序"——而这两种秩序，都不是天然给定的，都是在约束中被筛选出来的。
