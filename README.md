# 文明的原理 | Principles of Civilization

> **把象牙塔的黑箱，转译成普通人能理解的现实世界。**

---

## 这是什么

这是一个开放的知识转译工程。

现代文明积累了庞大的知识——物理、化学、生物、经济、金融、技术、历史……但这些知识越来越以专业系统内部的方式存在，离普通人的经验世界越来越远。

**文明的原理** 要做的事很简单：

把那些高密度、高门槛、高压缩的专业知识，重新翻译成普通人能够进入、理解、感受并逐步吸收的中高密度表达。

不是把知识变浅，而是把知识变得**可进入**。

---

## 核心理念

| 我们追求的 | 我们拒绝的 |
|-----------|-----------|
| 保留原理骨架 | 掏空结构讲故事 |
| 从人的经验入口进入 | 从术语定义起步 |
| 保留思想深度 | 为传播牺牲深度 |
| 中高密度可理解编码 | 低密度碎片化泛科普 |
| 可回译到严肃理论 | 与原理论断裂 |

一句话：**不是把复杂世界讲简单，而是把复杂世界重新翻译回人类。**

## 已有作品

### 《自然科学的哲学原理》

**母句**：科学不是知识的堆砌，而是有限存在者在误差可控中逼近可理解结构的持续努力。

| 卷 | 主题 | 状态 |
|----|------|------|
| 第一卷 | 变化：为什么一切都在动，万物凭什么可以被理解 | ✅ 定稿 |
| 第二卷 | 存在：为什么不是什么都能出现，现实的边界是怎样画出来的 | ✅ 定稿 |
| 第三卷 | 秩序：为什么趋向混乱的世界里，仍然存在稳定结构 | ✅ 定稿 |
| 第四卷 | 结合：为什么简单的东西会组成复杂的东西，涌现是什么意思 | ✅ 定稿 |
| 第五卷 | 生命：为什么有些东西能够自己维持自己，生命怎样从化学变成生物学 | ✅ 定稿 |
| 第六卷 | 知识：为什么知识永远是近似的，却仍然可靠 | ✅ 定稿 |

### 《经济学的哲学原理》

**母句**：经济不是金钱的流动，而是稀缺条件下，人类对生存机会、协作秩序与未来可能性的持续重组。

📋 规划中

## 三部曲的统一脊线

```
自然科学  →  解释世界如何存在      →  存在的结构
经济学    →  解释人群如何共存      →  协作的结构
金融学    →  解释文明如何穿越未来  →  时间的结构
```

> 从存在的结构，到协作的结构，再到时间的结构。

## 如何参与

这是一个**开放工程**。任何人都可以：

1. **贡献新的"XX的原理"** — 选一个领域，用同样的转译方法写作
2. **改进现有内容** — 提交 PR 修订文字、补充论证、增强可感性
3. **翻译** — 将中文原文翻译为其他语言
4. **讨论** — 在 Issues 中提出问题、质疑、建议

详见 [CONTRIBUTING.md](CONTRIBUTING.md)

## 多语言

本项目原文为中文。欢迎翻译为其他语言：

- 在对应领域目录下创建 `translations/` 子目录
- 文件名加语言后缀：如 `第一卷_变化.en.md`
- 翻译须保留原理骨架与结构，而非逐字直译
