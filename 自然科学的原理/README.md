# 自然科学的哲学原理

> **科学不是知识的堆砌，而是有限存在者在误差可控中逼近可理解结构的持续努力。**

---

## 总论

这部作品回答的不是"科学是什么"，而是：

**世界为什么能够被理解？人类这种有限的存在者，凭什么敢说自己知道？**

它把现代自然科学——从物理、化学到生物、认识论——重新翻译成普通人能够进入的世界图景。

---

## 卷目

| 卷 | 标题 | 核心问题 | 状态 |
|----|------|---------|------|
| 第一卷 | 变化：为什么一切都在动，万物凭什么可以被理解 | 运动与规律 | ✅ 定稿 |
| 第二卷 | 存在：为什么不是什么都能出现，现实的边界是怎样画出来的 | 约束与筛选 | ✅ 定稿 |
| 第三卷 | 秩序：为什么趋向混乱的世界里，仍然存在稳定结构 | 耗散与稳态 | ✅ 定稿 |
| 第四卷 | 结合：为什么简单的东西会组成复杂的东西，涌现是什么意思 | 层级与构型 | ✅ 定稿 |
| 第五卷 | 生命：为什么有些东西能够自己维持自己，生命怎样从化学变成生物学 | 自维持与演化 | ✅ 定稿 |
| 第六卷 | 知识：为什么知识永远是近似的，却仍然可靠 | 认知与近似 | ✅ 定稿 |

---

## 统一脊线

```
第一卷  变化  →  什么在动，如何看懂
第二卷  存在  →  什么能留下，什么被淘汰
第三卷  秩序  →  混乱中如何保住结构
第四卷  结合  →  简单如何变复杂
第五卷  生命  →  结构如何自维持
第六卷  知识  →  有限者如何逼近可理解现实
```

> 从变化到存在，从存在到秩序，从秩序到结合，从结合到生命，从生命到知识——同一条母线的逐层显影。

---

## 在整个系列中的位置

自然科学的原理处理的是**存在的结构**——世界如何存在，规律如何显影，理解如何可能。

它为后续的经济学（协作的结构）和金融学（时间的结构）奠定了自然哲学的地基。
